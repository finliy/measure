SliderComponent API Reference

Properties
Name Type Default Description
@Input()

input

boolean false If we can use input box to change the slider value, input value should be larger than min, less than max and multiples of the step. Only can use when range is false
@Input()

min

number 0

Min value of slider
@Input()

max

number 100

Max value of slider
@Input()

range

boolean false

If the slider has double hands
@Input()

disabled

boolean false

Whether the slider is disabled or not
@Input()

step

number 1

The step of dragging, should be divided by (max - min)
@Input()

value

number | number[] min Core value of slider
@Input()

orientation

string horizontal Orientation of slider, horizontal or vertical
@Output()

change

EventEmitter<CoreValue> The event emitted when slider value changes, emit the value of slider
