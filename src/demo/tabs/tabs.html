nb-tabs

type: Component

selector: nb-tabs

exportAs: nbTabs

Usage

import {TabsModule} from 'xdesign';

Examples

Tabs Basic

1.tabs可以设置 underline ，将在整体底部增加一条下划线作为标签和内容的分隔

2.tabs可以设置 splitline ，标签之间增加竖线作为分隔

3.tabs有两种尺寸 'default' | 'large'，默认情况下是较小尺寸，当显示大尺寸时设置 size="large"

4.tab可以设置 disabled ，用于不可使用状态

Tabs Tipable

标签上支持问号提示，在需要提示的tab上增加 tipable属性，并将提示内容用「 tip-content 」class名包裹起来

Tabs Content

内容支持任意HTML片段、Angular组件、指令，以嵌套tab组件为例说明
