nb-accordion

type: Component

selector: nb-accordion

exportAs: nbAccordion

Usage

import {AccordionModule} from 'xdesign';

Examples

Accordion Basic

1.支持 click 、hover 两种触发方式，默认情况下是 click，需要 hover 展开时设置 hoverable

2.折叠方式支持互斥折叠和全部可折叠，通过设置 collapsible， true 为全部可折叠， false 为互斥折叠
