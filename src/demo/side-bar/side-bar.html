nb-side-bar

type: Component

Usage

import {SideBarModule} from 'xdesign';

Examples

Basic

1.nb-side-bar用于展示树形结构的导航栏。

2.nb-side-bar有data属性，用于接收导航数据sideBarModel，包含title、expanded、root、tree项。

3.导航每个节点有selectable属性，置为false表示不可选择。

4.nb-side-bar有onNavi事件，用于将点中的导航栏节点的数据传出。
