nb-input

type: Directive

selector: input[nb-input]

exportAs: nb-input

Usage

import {InputModule} from 'xdesign';

Examples

Themes

1.用于设置输入框的主题颜色，有三种内置的 theme 样式，分别是：default, error, transparent。

2.如果没有显示指定，将使用默认的 default 样式。

3.如果设置了 disabled 属性，那么 disabled 样式会优先生效。

4.如果三种内置样式不符合你的需求，您还可以自定义主题，如上例，设置 theme="green"，会给输入框添加一个名为 nb-input-theme-green 的样式类。

Sizes

1.用于设置输入框的尺寸大小，有七种内置的 size 尺寸，从小到大分别是：long-high, long-middle, long-low, default, short-high, short-middle, short-low。

2.如果没有显示指定，将使用默认的 default 尺寸。

3.如果设置了 disabled 属性，那么 disabled 样式会优先生效。

4.如果七种内置尺寸不符合你的需求，您还可以自定义尺寸，如上例，设置 size="xs"，会给输入框添加一个名为 nb-input-size-xs 的尺寸样式类。
