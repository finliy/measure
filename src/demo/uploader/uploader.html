nb-uploader

type: Component

selector: nb-uploader

exportAs: nbUploader

Usage

import {UploaderModule} from 'xdesign';

Examples

Uploader Basic

1.对于文件的限制如下：

  • 文件的数量超过限制， 通过 maxFileCount="XXX" 设置， eg. maxFileCount="6"
  • 文件的大小不符合要求， 通过 maxFileSize="XXX" 设置， eg. maxFileSize="1000000"
  • 文件的类型不符合要， 通过 accept="XXX" 设置， eg. accept="image/*"

2.上传模式有图片上传和文件上传两种，通过 mode 设置，分别为 'image' | 'file'

3.上传进度有百分比和进度条两种，通过 progressMode设置，分别为 'text' | 'bar'

4.布局方式有水平和垂直两种，通过 direction设置，分别为 'horizontal' | 'vertical'

5.已上传的文件，通过 [files] 设置

6.自定义校验信息

  • 默认显示「文件的数量超过限制」， 可以通过 invalidFileCountMessage 自定义
  • 默认显示「文件的大小不符合要求」， 可以通过 invalidFileSizeMessage 自定义
  • 默认显示「文件的类型不符合要求」， 可以通过 invalidFileTypeMessage 自定义
  • 默认显示「上传失败」， 可以通过 uploadFailedMessage 自定义

6.通过 url设置

uploader ImageProgressBar

图片上传模式，上传进度使用进度条

uploader FileProgressText

文件上传模式，上传进度使用百分比显示

uploader FileProgressBar

文件上传模式，上传进度使用进度条

uploader FileVertical

垂直布局
