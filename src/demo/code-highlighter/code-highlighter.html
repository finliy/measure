nb-code-highlighter

type: Component

selector: nb-code-highlighter

Usage

import {CodeHighlighterModule} from 'xdesign';

Examples

code highlighter example

nb-code-highlighter 是一个代码高亮组件。可以使用 language 来指定语言， rawCode 来指定代码字符串，linenumbers 来设置是否显示代码行数。
