Checkbox Basic

1.单个选择框为 nb-checkbox，默认为复选框，有 checked，disabled，value，intermediate 等属性。

2.nb-checkbox 一般和 nb-boxgroup 一起使用，用来表示一个选择组，可以是单选组（type="radio"），也可以是多选组（type="checkbox"，默认）。

3.nb-boxgroup 的 value属性为一个字符串或者数字数组（不论是不是单选组）。

4.nb-checkbox 可以通过 theme 属性来定制样式。

5.nb-checkbox 可以通过 intermediate 属性来设置部分选择的中间状态。

Radiobox Basic

1. nb-checkbox 的 type属性设置成 'radio'，就变成了一个单选框。

2. 单个单选框使用没有意义，一般是用在 nb-boxgroup 内部，形成一个单选组。

Checkbox Group Basic

1.nb-boxgroup 内部使用 nb-checkbox 形成一个选择组，可以是单选组（type="radio"），也可以是多选组（type="checkbox"，默认）。

2.nb-boxgroup 的 value 属性为字符串或者数字数组，对应的 nb-checkbox 的 value 也必须为字符串或者数字。

3.nb-boxgroup 在值变更时会触发 change 事件，参数为包含 currentValue 和 value 的对象。

4.对于简单的选择框组，可以直接使用 nb-boxgroup 组件的 datasource 属性，其中每一项包括 value、text、checked 和 disabled 属性。

Radio Group Basic

1.nb-boxgroup 设置属性 type="radio"，可以变成单选框组。

2.单选框组的 value 属性也需要以数组形式提供。

box group with forms

nb-boxgroup 可以和 Forms 和 ReactiveForms 无缝结合使用。
