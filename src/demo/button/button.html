nb-button

type: Directive

selector: button[nb-button], a[nb-button]

exportAs: nbButton

Usage

import {ButtonModule} from 'xdesign';

Examples

Themes

1.用于设置按钮的主题颜色，有四种内置的 theme 样式，分别是：primary, default, neutral, transparent。

2.如果没有显示指定，将使用默认的 default 样式。

3.如果设置了 disabled 属性，那么 disabled 样式会优先生效。

4.如果四种内置样式不符合你的需求，您还可以自定义主题，如上例，设置 theme="yellow"，会给按钮添加一个名为 nb-button-theme-yellow 的样式类。

Sizes

1.用于设置按钮的尺寸大小，有四种内置的 size 尺寸，从小到大分别是：xs, sm, default, lg。

2.如果没有显示指定，将使用默认的 default 尺寸。

3.如果设置了 disabled 属性，那么 disabled 样式会优先生效。

4.如果四种内置尺寸不符合你的需求，您还可以自定义尺寸，如上例，设置 size="xxl"，会给按钮添加一个名为 nb-button-size-xxl 的尺寸样式类。

Link Button

您可以在 <a> 上添加 nb-button 属性，按钮呈现链接的形式。

Button with icons

1.由于按钮内容可以自定义，添加图标也很方便，可是使用内置的图标库，或者第三方的图标库，例如 fontawesome 等。

2.可以结合 theme 和 size 来实现圆形的图标按钮。
