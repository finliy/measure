nb-progress-bar, nb-progress-bar-circle

type: Component

Usage

import {ProgressBarModule} from 'xdesign';

Examples

Progress Bar Basic

1.nb-progress-bar 用直线的形式来显示进度。

2.属性 value 表示当前进度值，maxValue可以设置最大进度值（默认为100），还有一些设置文本显示的属性，详见下文API。

3.进度条有几种状态，isSuccess 为 true 时表示成功，isError 为 true 时表示失败。

Progress Circular Bar Basic

1.nb-progress-bar-circle 用甜甜圈的形式来显示进度。

2.属性 value 表示当前进度值，maxValue可以设置最大进度值（默认为100），还有一些设置文本显示的属性，详见下文API。

3.进度条有几种状态，isSuccess 为 true 时表示成功，isError 为 true 时表示失败。
