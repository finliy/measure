nb-carousel

type: Component

Usage

import {CarouselModule} from 'xdesign';

Examples

Carousel Basic

1.nb-carousel用来显示一系列轮播图片。

2.核心属性为 datasource，用于设置图片的来源和标题，另外还可以通过 width 设置轮播的宽度。

Carousel Auto

通过设置属性 auto 为 true，能够启动自动轮播。

Other Size Carousel

通过设置属性 skin 来设置不同的轮播外观， 例如设置成 'big' 后会生成 .nb-carousel-big 的样式类，你可以自行定义外观，默认提供 'big' 和 'small' 两种样式。

title navigation

通过设置 showTitle 属性为 true，可以在图片下方显示标题，同时启动标题导航功能。

arrow outside

通过设置 skin 为 'outside'，可以让导航箭头出现照片两侧，更加利于导航。
