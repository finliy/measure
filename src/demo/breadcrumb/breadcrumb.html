nb-breadcrumb

type: Component

selector: nb-breadcrumb

exportAs: xBreadcrumb

Usage

import {BreadcrumbModule} from 'xdesign';

Examples

Breadcrumb Examples

面包屑组件用来提示用户当前层级，以及进行快速导航，目前支持两种跳转模式，一种是 href 跳转（面包屑设置了 href 属性），另一种是 router 跳转（面包屑设置了 path 属性），详见例子。
