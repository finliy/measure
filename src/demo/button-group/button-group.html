nb-button-group, nb-button-toggle

type: Component

selector: nb-button-group，nb-button-toggle

exportAs: xButtonGroup，xButtonToggle

Usage

import {ToggleButtonModule} from 'xdesign';

Examples

Toggle Button Basic

1.单个 nb-button-toggle 类似 checkbox 和 switch，拥有 checked、disabled、value等属性，能够触发 toggle 事件。

2.更多的情况是，nb-button-toggle 作为 nb-button-group 组件的子组件，该组件有 type、value、disabled 等属性，能够触发 change 事件。

3.对于简单的按钮组，可以直接使用 nb-button-group 组件的 datasource 属性，其中每一项包括 value、text、checked 和 disabled 属性。

Toggle Button Group with (Reactive)Forms

nb-button-group 能够和 Forms 和 ReactiveForms 无缝结合使用。
