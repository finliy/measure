nb-search-box

type: Component

Usage

import {SearchBoxModule} from 'xdesign';

Examples

Basic

1.nb-search-box用于输入关键词去查询与关键词匹配的结果。

2.nb-search-box有type用于设置主题，有两种内置类型，分别是：ico, btn

3.nb-search-box有disabled 属性，用于置灰处理，为不可编辑状态。

4.nb-search-box有value属性，表示当前的关键词的值。

SearchBox with suggest

nb-search-box有isSuggestion属性，用于是否联想提示已输入的关键词结果。
