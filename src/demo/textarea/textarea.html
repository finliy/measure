nbTextarea

type: Directive

selector: textarea[nbTextarea]

exportAs: nbTextarea

Usage

import {TextareaModule} from 'xdesign';

Examples

Themes

1.用于设置文本输入框的主题颜色，有两种内置的 theme 样式，分别是：default, error。

2.如果没有显示指定，将使用默认的 default 样式。

3.如果设置了 disabled 属性，那么 disabled 样式会优先生效。

4.如果两种内置样式不符合你的需求，您还可以自定义主题，如上例，设置 theme="green"，会给文本输入框添加一个名为 nb-textarea-theme-green 的样式类。
