nb-transfer

type: Component

Usage

import {TransferModule} from 'xdesign';

Examples

Basic

1.nb-transfer用于实现左右列表选项状态互相转化。

2.nb-transfer左侧备选项和右侧已选项默认tree类型。

3.nb-transfer左侧备选项candidateData和右侧已选项selectedData用于接收展示左右两侧数据结构的数据，数据模型参考tree。

4.nb-transfer有disabled属性，用于置灰处理，为不可编辑状态。

5.nb-transfer有getValue事件，用于获取已选选项值，可以用于form表单。
